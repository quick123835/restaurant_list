## **安裝**
1. 打開終端機， Clone專案至本機電腦  
```
git clone https://github.com/quick123835/restaurant_list.git
```
2. 初始化

     
```
cd restaurant_list  // 切換至專案資料夾
```  
```
npm install // 安裝套件
```
3. 開啟程式
```
npm run dev
```
